## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Stack

- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Spline** (`@splinetool/react-spline`, `@splinetool/runtime`) — 3D scene
- **Radix UI** (`@radix-ui/react-slot`)
- **Lucide React** — icons
- **class-variance-authority**, **clsx**, **tailwind-merge**, **tailwindcss-animate**
